# AI Query Assistant Feature

## Overview

The AI Query Assistant is a feature extension to PocketBase that enables users to query database collections using natural language instead of learning PocketBase's filter syntax.

## Features

- Natural language query interface in Admin UI
- Schema-aware prompting (automatic field name understanding)
- Filter expression display with copy functionality
- API endpoint for programmatic access
- LLM configuration settings panel
- OpenAI integration (default)
- Basic validation and error handling
